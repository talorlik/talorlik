# Profile icons

Icons for skills that are not covered by [skill-icons](https://skillicons.dev) are loaded from the
[Simple Icons](https://simpleicons.org/) CDN in `docs/SKILLS_MANIFEST.yaml` (`icon_kind: simpleicons`).
See each skill row for the product link and tooltip metadata.

When adding a **local** SVG here, note the official source URL in the manifest or in this file.

- **openai.svg** - OpenAI mark geometry from [Simple Icons](https://github.com/simple-icons/simple-icons) (OpenAI), `fill` set to `#10A37F` (ChatGPT green) for contrast on both light and dark GitHub README themes; Skill Icons uses a near-black mark that vanishes on dark backgrounds.

- **cursor.svg** - Cursor mark from [Simple Icons](https://github.com/simple-icons/simple-icons) (Cursor), `fill` set to `#E6EDF3` so the logo stays visible on dark GitHub READMEs (the stock Simple Icons CDN color `#000000` blends into dark backgrounds).
